# Calendar Datepicker UI

### This project was used to showcase the basic implementations of grid and flex positioning, only CSS was used and no JavaScript :D

## Features:
* Uses HTML 5, and CSS only. JavaScript will come in hand in the future. 
* Implements **Grid Positioning**
* Includes a custom **Favicon**
* The datepicker has an input with a place holder, and grid positioned day boxes
* The project is pure CSS placeholders only, as functionality for this project will be implemented in future projects.

## How to Run:

This project can run locally within your computer without installation.

* Clone or Download this Repository.
    ```bash 
    git clone https://github.com/jFiesta11/Roadmap.sh-Frontend-Project-3---Datepicker-UI.git
    ```

* Double Click the `index.html` file to open in your default browser or right click the `index.html` file and click on 'open' or 'open with' and select the browser to open this file with.
